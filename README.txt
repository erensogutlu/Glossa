Glossa | Yazılım Öğrenme Platformu

Glossa, modern web teknolojilerini (JavaScript, React, Node.js, SQL) öğrenmek isteyen geliştiriciler için tasarlanmış bir eğitim platformudur. Tüm içerik ve kod yapısı tamamen Türkçe olarak hazırlanmıştır.

* Özellikler : 

 -> İnteraktif Dersler
 -> Bilgi Sayfaları
 -> Tamamen Türkçe
 -> Adım adım öğrenme deneyimi
 -> Mobil Uyumlu

* Kullanılan Teknolojiler :

   -> React
   -> Vite
   -> TailwindCSS

* Geliştirici : Eren Söğütlü

Glossa | Software Learning Platform

Glossa is a training platform designed for developers who want to learn modern web technologies (JavaScript, React, Node.js, SQL). All content and code structure are entirely in Turkish.

* Features :

 -> Interactive Lessons
 -> Information Pages
 -> Fully in Turkish
 -> Step-by-step learning experience
 -> Mobile Compatible

* Technologies Used :

 -> React
 -> Vite
 -> TailwindCSS

* Developer : Eren Söğütlü

Link : https://erensogutlu-glossa.netlify.app/

## Kurulum ve Çalıştırma

Bu proje Vite tabanlı bir React uygulamasıdır. Yerel ortamınızda çalıştırmak için aşağıdaki adımları izleyebilirsiniz :

### Gereksinimler
- Node.js (v18+)
- npm veya yarn

### Kurulum ve Çalıştırma
1. Bağımlılıkları yükleyin:
   ```bash
   npm install
   ```

2. Geliştirme sunucusunu başlatın:
   ```bash
   npm run dev
   ```

## Installation and Running

This project is a Vite-based React application. To run it in your local environment, you can follow these steps:

### Requirements
- Node.js (v18+)
- npm or yarn

### Installation and Running
1. Install dependencies:
```bash
npm install
```
2. Start the development server:
```bash
npm run dev
```
